# WSL
WSL is awesome. Install it by running `wsl.exe --install` in Powershell.

To install a specific distribution, use `wsl.exe --install distribution`

View your distributions with `wsl --list --verbose`

Unregister a distribution using `wsl --unregister distribution`

Update the current distribution with `wsl --update`

Install C/C++ essentials with `sudo apt-get install gcc`, `sudo apt-get install g++`, and `sudo apt-get install make`

Install the WSL extension in VSCode for better IDE support.

Open code in VSCode using `code .`
